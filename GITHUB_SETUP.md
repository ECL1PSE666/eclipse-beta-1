# Инструкции по созданию репозитория на GitHub

## Шаг 1: Создание репозитория на GitHub

1. Перейдите на [github.com](https://github.com) и войдите в аккаунт
2. Нажмите на кнопку "+" в правом верхнем углу и выберите "New repository"
3. Заполните форму:
   - **Repository name**: `eclipse-beta-test`
   - **Description**: `Eclipse - современная видеоплатформа на Next.js и Supabase`
   - **Public**: ✅ (выберите публичный)
   - **Initialize repository**: ❌ (НЕ ставьте галочки, так как у нас уже есть код)
4. Нажмите "Create repository"

## Шаг 2: Подключение локального репозитория

После создания репозитория GitHub покажет страницу с инструкциями. Выполните следующие команды в терминале:

```bash
git remote add origin https://github.com/ECL1PSE666/eclipse-beta-test.git
git branch -M main
git push -u origin main
```

## Шаг 3: Проверка

После выполнения команд репозиторий будет создан и весь код будет загружен на GitHub.

## Альтернативный способ (если нужно)

Если предпочитаете использовать веб-интерфейс для загрузки:

1. На странице созданного репозитория нажмите "uploading an existing file"
2. Перетащите все файлы из папки проекта
3. Добавьте коммит: "Initial commit: Eclipse video platform"
4. Нажмите "Commit changes"

## Что после?

После успешной загрузки:
- Ваш проект будет доступен по адресу: https://github.com/ECL1PSE666/eclipse-beta-test
- README.md автоматически отобразится на главной странице
- Проект будет публично доступен для других разработчиков
